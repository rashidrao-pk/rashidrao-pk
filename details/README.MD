I am a PhD researcher at the **University of Turin, Italy**, working on **Computer Vision**, **eXplainable AI (XAI)**, and **visual anomaly detection** for industrial applications.

My recent work focuses on:
- explainable computer vision
- anomaly detection
- feature attribution methods
- trustworthy AI
- image-based machine learning systems



## Research Highlights

- **ShapBPT (AAAI 2026)**  
  Image feature attributions using data-aware binary partition trees  
  - [Paper](https://ojs.aaai.org/index.php/AAAI/article/view/39699), [arXiv](https://arxiv.org/abs/2602.07047), [Code](https://github.com/amparore/shap_bpt), [Tests](https://github.com/rashidrao-pk/shap_bpt_tests), [PyPI](https://pypi.org/project/shap-bpt/)

- **Using Stratified Sampling to Improve LIME Image Explanations (AAAI 2024)**  
  - [Paper](https://ojs.aaai.org/index.php/AAAI/article/view/29397), [Code](https://github.com/rashidrao-pk/lime_stratified), [Examples](https://github.com/rashidrao-pk/lime-stratified-examples), [PyPI](https://pypi.org/project/lime-stratified/)

- **Can I Trust My Anomaly Detection System? A Case Study Based on Explainable AI (XAI-World 2024)**  
  - [Paper](https://link.springer.com/chapter/10.1007/978-3-031-63803-9_13), [Code](https://github.com/rashidrao-pk/anomaly_detection_trust_case_study)
